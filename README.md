# 一丶简述

#### 实现功能
1. 手动发送弹幕
2. 定时发送弹幕
3. 接收弹幕
4. 根据礼物感谢
5. 欢迎老爷舰长进入房间
6. 接入图灵机器人可以聊天
7. 钉钉群推送
